# Dual-Modal OCR Experiment

本目录用于存放“视觉特征 + 文本语义特征”双模态 OCR 改进实验的辅助说明与配置样例。

## 设计目标

- 保留现有 baseline 作为对照组
- 新增独立训练与测试链路
- 在识别头之前插入语义辅助分支和门控融合模块

## 当前实现约定

- 视觉主模态：单行文本图像
- 文本语义模态：字符序列编码后的语义向量
- 训练阶段：使用真实标签构建语义分支
- 推理阶段：使用视觉分支初步预测构建语义分支

## 主要脚本

- `scripts/09_train_dual_modal.py`
- `scripts/10_predict_dual_modal.py`
